# Image organizer for word
This Python script sorts images from a folder by their creation date and automatically inserts them into a Word document in the correct chronological order.

## Purpose
When multiple images are copied and then manually inserted into Word, they are not ordered correctly. This script solves that problem and saves time.

## Built with
- Python
- python-docx
- Pillow

## Usage
1. Enter the path of the folder that contains the images
2. Enter a name for the Word document that is going to be created
3. Enter the height of the images
4. A word document is created and saved in the folder of the images
